# 服务框架说明：
| **技术** |  **名称**| 
| ------   |:------:|
| Spring Framework | 容器  |
| SpringMVC | MVC框架  |
| Spring Boot|构建系统|
| Apache Shiro | 安全框架  |
| MyBatis | ORM框架  |
| MyBatis Generator | 代码生成  |
| PageHelper | MyBatis物理分页插件  |
| Druid | 数据库连接池  | 
| Redis | 分布式缓存数据库  |
| Log4J | 日志组件  | 
| Fastjson |JSON处理器|
| Maven | 项目构建管理  | 
| dubbo|分布式服务框架|
